# CODSOFT Python Internship

This repository contains the Python projects completed during my CODSOFT Python Programming Internship.

## Projects

### 1. Calculator
A GUI calculator built using Python and Tkinter.

Features:
- Addition
- Subtraction
- Multiplication
- Division
- Keyboard support

### 2. To-Do List
A task management application.

Features:
- Add tasks
- Delete tasks
- Save tasks

### 3. Contact Book
A contact management system.

Features:
- Add contacts
- Search contacts
- Update contacts
- Delete contacts

## Technologies Used

- Python
- Tkinter
- SQLite
